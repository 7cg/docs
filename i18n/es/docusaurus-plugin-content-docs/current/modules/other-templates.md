# Módulo Otras Plantillas

El módulo **Otras Plantillas** es el panel genérico de 7CG para plantillas que no encajan en los flujos principales de Tercios Inferiores, Biblia, Letras o Créditos.

Ofrece:

- Un selector de plantilla sencillo
- Controles rápidos de Reproducir y Detener
- Un lugar dedicado en el espacio de trabajo para plantillas adicionales o diversas

## Visión general

El módulo Otras Plantillas es útil cuando necesitas un espacio ligero para gráficos que no encajan en uno de los módulos más especializados.

Ejemplos típicos:

- Gráficos diversos
- Superposiciones puntuales
- Gráficos temporales para un evento
- Plantillas utilitarias que no necesitan un flujo dedicado

Es intencionadamente simple. Considéralo un panel flexible para plantillas extra más que un sistema completo de edición.

## Componentes de la interfaz

### Selector de plantilla

El módulo incluye un desplegable para elegir entre las entradas disponibles en el panel.

Por ahora, el selector es muy básico. No se presenta como un navegador dinámico completo de plantillas, a diferencia de los módulos gráficos más avanzados.

### Controles Reproducir y Detener

El módulo incluye:

- **Reproducir**
- **Detener**

Establecen el flujo de lanzamiento rápido previsto para plantillas diversas.

## Estado actual del módulo

El módulo Otras Plantillas está presente en el espacio de trabajo, pero su implementación actual es mínima en comparación con los módulos más maduros.

Lo que ya ofrece:

- Un módulo dedicado en el diseño
- Un desplegable de plantillas
- Botones básicos de Reproducir y Detener
- Un lugar cómodo para gráficos extra fuera de los módulos especializados

Lo que aún no ofrece como flujo más rico:

- Un editor de campos completo guiado por metadatos
- Controles avanzados de enrutamiento en el propio módulo
- Acciones de añadir al rundown
- Un flujo de preparación especializado como el de Tercios Inferiores, Biblia, Letras o Créditos

Por eso, lee esta página como orientación para un módulo utilitario simple y no como un flujo completo de producción gráfica.

## Cuándo usarlo

Usa Otras Plantillas cuando:

- El gráfico no encaje en un módulo dedicado
- Necesites un hueco rápido para una plantilla extra en el espacio de trabajo
- La producción incluya un número pequeño de superposiciones diversas

Evita usarlo como sustituto de módulos más especializados cuando ya existe un flujo dedicado. Por ejemplo:

- Usa **Tercios Inferiores** para gráficos de nombre/título
- Usa **Biblia** para pasajes bíblicos
- Usa **Letras** para himnos y letras de canciones
- Usa **Créditos** para agradecimientos de producción y rollos finales

## Flujo típico

1. Abre el módulo Otras Plantillas
2. Elige la plantilla a usar
3. Haz clic en **Reproducir** para ponerla al aire
4. Haz clic en **Detener** cuando ya no se necesite

## Buenas prácticas

- Usa este módulo para gráficos diversos, no para los que ya tienen un módulo dedicado
- Mantén expectativas simples: es un panel de acceso rápido, no un flujo completo de autoría
- Coloca el módulo en un diseño donde los gráficos utilitarios sean fáciles de alcanzar sin distraer de los controles principales
- Usa una nomenclatura clara para que los operadores reconozcan rápidamente las entradas
