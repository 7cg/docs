# ¿Qué es 7CG?

**7CG** es una aplicación profesional de control de gráficos para transmisión, diseñada específicamente para servidores **CasparCG**. Pensada para iglesias, emisoras y equipos de producción, 7CG ofrece herramientas potentes para gestionar gráficos en vivo, reproducción de medios y presentación de contenido durante cultos y producciones en directo.

## ¿Para quién es 7CG?

7CG se creó pensando en las **iglesias y ministerios de comunicación Adventistas del Séptimo Día**, pero sirve a cualquier organización que necesite control profesional de gráficos para transmisión:

- **Iglesias** que transmiten cultos en vivo o grabados
- **Equipos de producción de medios** que gestionan gráficos y contenido para eventos en vivo
- **Profesionales de transmisión** que controlan servidores de gráficos CasparCG
- **Directores técnicos** que coordinan producciones visuales con múltiples capas

## Características principales

### Gestión del Rundown
Planifica y ejecuta todo el programa con precisión. El sistema de rundown permite crear secuencias de bloques, ejecutar la selección actual, disparar elementos específicos desde Companion y exportar entradas compatibles a vídeo.

### Tercios inferiores y gráficos
Muestra tercios inferiores profesionales con nombres, títulos y gráficos personalizados. Compatibilidad total con plantillas de CasparCG con transiciones, animaciones y diseños compatibles con GDD.

### Biblia y Escrituras
Base de datos bíblica integrada y multilingüe, con soporte para:
- Varias traducciones
- Visualización versículo por versículo
- Búsqueda y navegación
- Formato y plantillas personalizadas

### Himnario y letras
Sistema de himnario integrado para mostrar letras de canciones:
- Visualización versículo por versículo o por grupos
- Varios himnarios
- Compatibilidad con canciones personalizadas
- Sincronización con plantillas gráficas

### Reproducción de medios
Controla la reproducción de vídeo, imagen y audio con:
- Efectos de transición (mix, wipe, slide, etc.)
- Enrutamiento por canal y capa
- Controles de tiempo y retardo
- Gestión de salida en preview y programa

### Espacios de trabajo flexibles
Personaliza el espacio de trabajo con módulos movibles, columnas redimensionables, módulos ocultos y diseños guardados con nombre para diferentes operadores, producciones o lugares.

### Emparejamiento y descubrimiento con Companion
Empareja dispositivos Bitfocus Companion con códigos PIN de corta duración, gestiona cada superficie de control de forma independiente y aprovecha el descubrimiento en la red local cuando esté disponible.

### Soporte multilingüe
Localización completa de la interfaz en inglés, portugués y español, con soporte para mostrar contenido en varios idiomas.

## Casos de uso comunes

### Gráficos para cultos
Muestra nombres de oradores, versículos bíblicos, letras de himnos y anuncios en una secuencia coordinada, controlada desde una única interfaz.

### Producción de eventos en vivo
Gestiona rundowns complejos con reproducción de medios, superposiciones gráficas y efectos de transición para conferencias, conciertos o eventos especiales.

### Transmisiones con varias cámaras
Controla los gráficos en varios canales de CasparCG para diferentes salidas (programa, preview, streaming, grabación).

### Secuencias automatizadas
Crea secuencias automatizadas de bloques con tiempo y transiciones para una ejecución sin intervención durante momentos críticos.

### Operación por roles
Crea diseños distintos para operadores de gráficos, operadores de letras o directores técnicos, de modo que cada usuario vea el conjunto adecuado de herramientas para su tarea.
