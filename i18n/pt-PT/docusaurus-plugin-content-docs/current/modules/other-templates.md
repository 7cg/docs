# Módulo Outros Templates

O módulo **Outros Templates** é o painel genérico do 7CG para templates que não pertencem aos fluxos principais de Oráculos, Bíblia, Louvor ou Ficha Técnica.

Fornece:

- Um seletor de templates simples
- Controlos rápidos de Reproduzir e Parar
- Um local dedicado no espaço de trabalho para templates extras ou diversos

## Visão geral

O módulo Outros Templates é útil quando precisa de um local leve para gráficos que não encaixam num dos módulos mais especializados.

Exemplos típicos:

- Gráficos diversos
- Sobreposições pontuais
- Gráficos temporários de evento
- Templates utilitários que não precisam de um fluxo dedicado

É intencionalmente simples. Veja-o como um painel flexível para templates extras, não como um sistema completo de edição.

## Componentes da interface

### Seletor de template

O módulo inclui uma lista pendente para escolher entre as entradas atualmente disponíveis no painel.

De momento, este seletor é muito básico. Não se apresenta como um navegador dinâmico completo de templates, ao contrário dos módulos gráficos mais avançados.

### Controlos Reproduzir e Parar

O módulo inclui:

- **Reproduzir**
- **Parar**

Estabelecem o fluxo de lançamento rápido pretendido para templates diversos.

## Estado atual do módulo

O módulo Outros Templates está presente no espaço de trabalho, mas a implementação atual é mínima comparada com os módulos mais maduros.

O que já fornece:

- Um módulo dedicado no layout
- Uma lista pendente de templates
- Botões básicos de Reproduzir e Parar
- Um local conveniente para gráficos extra fora dos módulos especializados

O que ainda não oferece como fluxo mais rico:

- Um editor de campos completo guiado pelos metadados
- Controlos avançados de encaminhamento no próprio módulo
- Ações de adicionar ao rundown
- Um fluxo de preparação especializado como o usado por Oráculos, Bíblia, Louvor ou Ficha Técnica

Por isso, leia esta página como orientação para um módulo utilitário simples, não como um fluxo completo de produção gráfica.

## Quando usar

Use Outros Templates quando:

- O gráfico não cabe num módulo dedicado
- Precisa de um espaço rápido para um template extra no workspace
- A produção inclui um pequeno número de sobreposições diversas

Evite usá-lo como substituto de módulos mais especializados quando existe um fluxo dedicado. Por exemplo:

- Use **Oráculos** para gráficos de nome/título
- Use **Bíblia** para passagens bíblicas
- Use **Louvor** para hinos e letras de canções
- Use **Ficha Técnica** para créditos de produção e rolos finais

## Fluxo típico

1. Abra o módulo Outros Templates
2. Escolha o template a usar
3. Clique em **Reproduzir** para o colocar no ar
4. Clique em **Parar** quando já não for necessário

## Boas práticas

- Use este módulo para gráficos diversos, não para os que já têm um módulo dedicado
- Mantenha expectativas simples: é um painel de acesso rápido, não um fluxo completo de autoria de templates
- Coloque o módulo num layout onde os gráficos utilitários sejam fáceis de alcançar sem distrair dos controlos principais
- Use uma nomenclatura clara para que os operadores reconheçam rapidamente as entradas
