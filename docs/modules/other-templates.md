# Other Templates Module

The **Other Templates** module is 7CG's catch-all panel for templates that do not belong in the main Lower Thirds, Bible, Lyrics, or Credits workflows.

It provides:

- A simple template selector
- Quick Play and Stop controls
- A dedicated place in the workspace for extra or miscellaneous templates

## Overview

The Other Templates module is useful when you need a lightweight place for graphics that do not fit one of the more specialized modules.

Typical examples include:

- Miscellaneous graphics
- One-off overlays
- Temporary event graphics
- Utility templates that do not need a dedicated workflow

This module is intentionally simple. It is best understood as a flexible extra-template panel rather than a fully developed template-editing system.

## Interface Components

### Template Selector

The module includes a dropdown for choosing from the currently available entries in the panel.

At the moment, this selector is very basic. It is not presented as a full dynamic template browser like the more advanced graphics modules.

### Play and Stop Controls

The module includes:

- **Play**
- **Stop**

These controls establish the intended quick-launch workflow for miscellaneous templates.

## Current State of the Module

The Other Templates module is present in the workspace, but its current implementation is minimal compared with the more mature modules in 7CG.

What it already provides:

- A dedicated module in the layout
- A template dropdown
- Basic Play and Stop buttons
- A convenient place for extra graphics outside the main specialized modules

What it does not currently offer as a richer workflow:

- A full metadata-driven field editor
- Advanced routing controls in the module itself
- Add-to-rundown actions
- A specialized preparation flow like the one used by Lower Thirds, Bible, Lyrics, or Credits

Because of that, this page should be read as guidance for a simple utility module rather than a complete graphics production workflow.

## When to Use It

Use Other Templates when:

- The graphic does not fit a dedicated module
- You need a quick extra template slot in the workspace
- The production includes a small number of miscellaneous overlays

Avoid using it as a replacement for more specialized modules when a dedicated workflow already exists. For example:

- Use **Lower Thirds** for name/title graphics
- Use **Bible** for scripture passages
- Use **Lyrics** for hymns and song lyrics
- Use **Credits** for production acknowledgments and end rolls

## Typical Workflow

1. Open the Other Templates module
2. Choose the template you want to use
3. Click **Play** to put it on air
4. Click **Stop** when it is no longer needed

## Best Practices

- Use this module for miscellaneous graphics, not for graphics that already have a dedicated module
- Keep expectations simple: this is a quick-access panel, not a full template authoring workflow
- Place the module in a workspace layout where utility graphics are easy to reach but do not distract from core show controls
- Use clear internal template naming so operators can recognize entries quickly
