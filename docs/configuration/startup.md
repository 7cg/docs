# Startup

Configure application behavior when 7CG launches.

## Startup Options

**Bug Graphics**
- Display bug/logo graphics on startup
- Useful for automatically showing station identification

**Station ID**
- Display station identification on startup
- Automatically show channel branding

:::note
Startup behavior options are planned features. Check the current version's capabilities in the About tab.
:::
