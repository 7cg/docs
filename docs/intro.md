# What is 7CG?

**7CG** is a professional broadcast graphics control application built specifically for **CasparCG servers**. Designed for churches, broadcasters, and production teams, 7CG provides powerful tools to manage live graphics, media playback, and content display during worship services and live productions.

## Who is 7CG For?

7CG was created with **Seventh-day Adventist churches and media ministries** in mind, but serves any organization that needs professional broadcast graphics control:

- **Churches** running live-streamed or broadcast worship services
- **Media production teams** managing graphics and content for live events
- **Broadcast professionals** controlling CasparCG graphics servers
- **Technical directors** coordinating multi-layered visual productions

## Key Features

### Rundown Management
Plan and execute your entire show with precision. The rundown system lets you create sequences of blocks, execute the current selection, target specific items from Companion, and export supported rundown entries to video.

### Lower Thirds & Graphics
Display professional lower thirds with names, titles, and custom graphics. Full support for CasparCG templates with transitions, animations, and GDD-compliant designs.

### Bible & Scripture
Integrated multi-language Bible database with support for:
- Multiple translations
- Verse-by-verse display
- Search and navigation
- Custom formatting and templates

### Hymnal & Lyrics
Built-in hymnal system for displaying song lyrics:
- Verse-by-verse or multi-verse display
- Multiple songbooks
- Support for custom songs
- Synchronized with graphics templates

### Media Playback
Control video, image, and audio playback with:
- Transition effects (mix, wipe, slide, etc.)
- Channel and layer routing
- Timing and delay controls
- Preview and program output management

### Flexible Operator Workspaces
Customize the workspace with movable modules, resizable columns, hidden modules, and named layout presets for different operators, productions, or venues.

### Companion Pairing & Discovery
Pair Bitfocus Companion devices with short-lived PIN codes, manage each control surface independently, and take advantage of local-network discovery where supported.

### Multi-Language Support
Full interface localization in English, Portuguese, and Spanish, with support for multi-language content display.

## Common Use Cases

### Worship Service Graphics
Display speaker names, Bible verses, song lyrics, and announcements in a coordinated sequence controlled from a single interface.

### Live Event Production
Manage complex rundowns with media playback, graphics overlays, and transition effects for conferences, concerts, or special events.

### Multi-Camera Broadcasts
Control graphics across multiple CasparCG channels for different outputs (program, preview, streaming, recording).

### Automated Sequences
Create automated block sequences with timing and transitions for hands-free execution during critical moments.

### Role-Based Operation
Build different layouts for graphics operators, lyrics operators, or technical directors so each user sees the right set of tools for the job.
