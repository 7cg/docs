Interface

Personalize a aparência visual e os módulos visíveis aos operadores no espaço de trabalho principal.

Tema​

Escolha o esquema de cores preferido:

  • Escuro — Tema escuro otimizado para ambientes com pouca luz
  • Claro — Tema claro para ambientes muito iluminados
  • Sistema — Segue a preferência de tema do sistema operativo

:::tip Boa prática A maioria dos ambientes de transmissão usa o tema Escuro para reduzir o cansaço visual em produções longas. :::

Idioma​

Selecione o idioma preferido para a interface:

  • English (en)
  • Português (pt)
  • Español (es)

As alterações de idioma aplicam-se de imediato a todos os elementos da interface, incluindo:

  • Itens de menu e botões
  • Caixas de diálogo
  • Mensagens de estado
  • Etiquetas de formulário

Módulos​

Controle a visibilidade de módulos e funcionalidades específicas:

  • Multimédia — Reprodução de vídeo, imagem e áudio
  • Louvor — Letras e sistema de hinário
  • Hino — Módulo dedicado a blocos de hino
  • Bíblia — Apresentação de Escrituras
  • Oráculos — Gráficos de nome e título (lower thirds)
  • QR Code — Blocos de gerador de QR Code
  • Ficha Técnica — Créditos de produção e agradecimentos
  • Ticker — Texto rolante
  • Info Canal — Gráficos específicos do canal
  • Camadas — Vista em direto das camadas CasparCG em uso
  • Emergência — Sistema de alertas de emergência

As alterações de visibilidade aplicam-se de imediato e são úteis para simplificar a interface para um determinado operador ou evento.

Como a visibilidade dos módulos se relaciona com os layouts​

A visibilidade dos módulos e a personalização do layout estão relacionadas, mas não são a mesma coisa:

  • Interface → Módulos controla se um módulo está sequer disponível
  • Layouts controla onde os módulos visíveis são colocados, ocultados temporariamente ou agrupados em predefinições guardadas

Se precisar de reorganizar colunas, ocultar módulos temporariamente ou guardar espaços de trabalho específicos por função, consulte Layouts.

Recomendações por operador​

Algumas configurações comuns:

  • Operador de letras — Manter Louvor, Bíblia, Rundown e Multimédia visíveis
  • Operador de gráficos — Manter Oráculos, Info Canal, Ficha Técnica e Rundown visíveis
  • Diretor técnico — Manter a maioria dos módulos visíveis e usar predefinições de layout para diferentes fases do programa

Notas​

  • As alterações de tema aplicam-se em toda a aplicação
  • As alterações de idioma são imediatas
  • A visibilidade dos módulos é guardada por instalação e mantém-se entre arranques
